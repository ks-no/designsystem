# Designsystem for KS Digital

KS Digital lager en rekke ulike løsninger for kommunene.

Designsystemet er limet som får disse løsningene til å følge samme standarder.
Standardene er viktig for oss. Det gir forutsigbar oppførsel, konsistens utseende og universell utforming.
Det gjør det enklere for oss å vedlikeholde og videreutvikle løsningene over tid.

## Filosofi

Vi bruker komponenter direkte der det er mulig og lager egne komponenter kun når det er behov for spesialtilpasninger til våre løsninger. Vi prioriterer rammeverksuavhengige komponenter for å støtte ulike teknologier.

## For designere

Designsystemet er tilgjengelig som et bibliotek i Figma. Der kan du utforske de ulike temaene og komponetene som er tilgjengelig.

## For utviklere

1. Installer tema-pakken og ta i bruk temaet som passer løsningen din.

```
pnpm add @ks-digital/designsystem-themes
```

2. Tema-pakken fungerer sammen med `react` og `web`-pakkene til Digdir. Du trenger ikke innstallere `css`-pakken.

3. Innstaller web-pakken for å bruke KS Digital sine egne komponenter.

```
pnpm add @ks-digital/designsystem-web
```
