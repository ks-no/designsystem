# React-pakken

1-til-1-implementering av komponentene som finnes sentralt hos Digdir, derfor har vi foreløpig ikke laget egen dokumentasjon. Sjekk ut [dokumentasjonen hos Digdir](https://designsystemet.no/no/components) for de ulike komponentene. ☺️

## Installasjon

```bash
pnpm add @ks-digital/designsystem-react
```

## Bruk

1. Hvis du ikke allerede har gjort det, sett opp et [tema](/docs/temaer--docs) fra `@ks-digital/designsystem-themes`-pakken.

2. Importer og bruk komponentene du trenger

```tsx
import { Button } from '@ks-digital/designsystem-react'

function MyComponent() {
  return <Button data-size="lg">My Button</Button>
}
```
