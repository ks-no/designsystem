# Suggestion

<OfficialDocsLink href="https://designsystemet.no/no/components/docs/suggestion/overview" />

`Suggestion` er en wrapper for `ds-suggestion` web-componenten fra Designsystemet for å gjøre den litt enklere å bruke i Angular-prosjekter.

<Primary />
<Controls />

## Bruk

- Bruk `selected` til å lagre nåværende valgt verdi (eller verdier ved `multiple`).
- Lytt på `selectedChange` for å reagere når brukeren velger eller fjerner et valg.

```ts
import { Component, signal } from '@angular/core'
import { Field, Input, Label } from '@ks-digital/designsystem-angular/forms'
import {
  Suggestion,
  type SuggestionItem,
  SuggestionList,
  SuggestionListEmpty,
  SuggestionListOption,
} from '@ks-digital/designsystem-angular/suggestion'

@Component({
  imports: [
    Field,
    Label,
    Input,
    Suggestion,
    SuggestionList,
    SuggestionListEmpty,
    SuggestionListOption,
  ],
  template: `
    <ksd-field>
      <ksd-label>Velg destinasjon</ksd-label>
      <ksd-suggestion
        [selected]="selected()"
        (selectedChange)="selected.set($event)"
      >
        <input ksd-input />
        <del aria-label="Tøm" hidden=""></del>
        <ksd-suggestion-list>
          @for (place of places(); track place) {
            <ksd-suggestion-list-option [value]="place">{{
              place
            }}</ksd-suggestion-list-option>
          }
          <ksd-suggestion-list-empty>Ingen treff</ksd-suggestion-list-empty>
        </ksd-suggestion-list>
      </ksd-suggestion>
    </ksd-field>
  `,
})
export class App {
  places = signal(['Bergen', 'Oslo'])
  selected = signal<SuggestionItem | undefined>(undefined)
}
```

### Eget filter

Om du ønsker å implementere eget filter, kan du gjøre det ved å lytte på `input`-eventen på input-feltet og filtrere listen av forslag selv. Sett `[filter]="false"` på `ksd-suggestion` for å deaktivere det innebygde filteret.
