# Angular-pakken

Et tynt wrapper-lag oppå Digdir sin web-pakke, laget for å forenkle integrasjon med Angular-applikasjoner. Pakken ble opprinnelig utviklet før Digdirs web-pakke var tilgjengelig. Vi vurderer fortløpende om den gir tilstrekkelig merverdi til å opprettholdes, ettersom Angular i dag har god støtte for web components.

## Filosofi

- Basert utelukkende på moderne, standalone Angular-komponenter med signals for state-håndtering.
- Lightweight, ingen eksterne avhengigheter utenom `@digdir/designsystemet-web`.
- Separate entrypoints. Importér bare det du trenger.
- Utvidet med typing for data-attributter.
- Støtter kun Angular v20 og nyere.
- Dokumentasjon for best practices ved bruk av komponentene finnes primært hos Digdir for å slippe dobbelt vedlikehold. Hver story har en lenke til offisiell dokumentasjon.
- Pakken inneholder ingen egne komponenter utover de som allerede finnes hos Digdir Designsystemet. Sammensatte komponenter lages i web-pakken for å fungere på tvers av rammeverk.
- API-et kan endres uten forvarsel, men vi vil lage migreringer for interne løsninger. Bruk på eget ansvar dersom du benytter pakken utenfor KS Digital.

## Installasjon

```bash
pnpm add @ks-digital/designsystem-angular
```
