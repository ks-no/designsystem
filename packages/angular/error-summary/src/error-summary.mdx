# ErrorSummary

`ErrorSummary` er en oppsummering av feil. Den gir brukeren oversikt over feil eller mangler som må rettes på en side eller trinn, for å komme videre.

<Primary />
<Controls />

## Bruk

```html
<div ksd-error-summary>
  <h2 ksd-heading>For å gå videre må du rette opp følgende feil:</h2>
  <ul class="ds-list">
    <li>
      <a ksd-link data-color="neutral" href="#field-id">
        Feilmelding som matcher feltets feilmelding
      </a>
    </li>
  </ul>
</div>
```

## Retningslinjer og eksempler

For utfyllende retningslinjer, tilgjengelighetsinformasjon og flere eksempler, se [ErrorSummary på designsystemet.no](https://designsystemet.no/no/components/docs/error-summary/overview).

### Viktige punkter

- Bruk `ErrorSummary` når et skjema har mange felt hvor det kan være vanskelig å få oversikt over feilene
- Vis komponenten først når brukeren prøver å sende inn skjemaet
- Komponenten skal få fokus når den vises. `tabindex=-1` blir automatisk satt på `ksd-error-summary`, og komponenten har en public `focus()`-metode.
- Feilmeldingene skal lenke direkte til feltet med feil
- Feilmeldingene i oppsummeringen skal være formulert likt som feilmeldingen ved feltet
