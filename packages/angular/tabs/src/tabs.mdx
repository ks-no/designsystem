# Tabs

<OfficialDocsLink href="https://designsystemet.no/no/components/docs/tabs/overview" />

`Tabs` lar brukerne navigere mellom relaterte deler av innholdet, der én del vises om gangen. Dette er en effektiv måte å organisere og presentere relatert innhold på samme side.

<Primary />
<Controls />

## Bruk

```ts
import {
  Tabs,
  TabsList,
  TabsTab,
  TabsPanel,
} from '@ks-digital/designsystem-angular'

@Component({
  imports: [Tabs, TabsList, TabsTab, TabsPanel],
  template: `
    <ksd-tabs>
      <ksd-tabs-list>
        <ksd-tabs-tab>Tab 1</ksd-tabs-tab>
        <ksd-tabs-tab>Tab 2</ksd-tabs-tab>
        <ksd-tabs-tab>Tab 3</ksd-tabs-tab>
      </ksd-tabs-list>
      <ksd-tabs-panel>Content 1</ksd-tabs-panel>
      <ksd-tabs-panel>Content 2</ksd-tabs-panel>
      <ksd-tabs-panel>Content 3</ksd-tabs-panel>
    </ksd-tabs>
  `,
})
export class App {}
```

## Reagere på endring

Bruk `tabClicked`-outputen for å lytte etter klikk på tabs. Du kan også sette `tabId` for å enklere identifisere hvilken tab som er klikket på.

```ts
@Component({
  imports: [Tabs, TabsList, TabsTab, TabsPanel],
  template: `
    <ksd-tabs>
      <ksd-tabs-list>
        <ksd-tabs-tab tabId="users" (tabClicked)="onTabClicked($event)">
          Tab 1
        </ksd-tabs-tab>
      </ksd-tabs-list>
      <ksd-tabs-panel>Content 1</ksd-tabs-panel>
    </ksd-tabs>
  `,
})
export class App {}
```
