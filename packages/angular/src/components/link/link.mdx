# Link

`Link` er klikkbar tekst eller grafikk, som gjør det enkelt for brukerne å navigere til filer eller sider internt eller eksternt. Komponenten er basert på [anchor HTML-taggen (mozilla.org)](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a).

<Canvas of={LinkStories.Preview} />

## Bruk

```ts
import { Link } from '@ks-digital/designsystem-angular';

<a ksd-link href="https://designsystemet.no/">
  Gå til designsystemet
</a>
```
