# Card

Med Card kan vi fremheve informasjon eller oppgaver som hører sammen.

<Primary />
<Controls />

## Bruk

`ksd-card` brukes som direktiv sammen med riktig HTML-tag i din kontekst, f.eks `article`. Du kan bruke `ksd-card-block` for å dele kortet inn i flere seksjoner.

```ts
import { Card, CardBlock } from '@ks-digital/designsystem-angular/card'

@Component({
  imports: [Card, CardBlock],
  template: `
    <article ksd-card>
      <h2 ksd-card-block>Tittel</h2>
      <p ksd-card-block>Innhold</p>
      <p ksd-card-block>Valgfri fotnote</p>
    </article>
  `,
})
export class App {}
```
