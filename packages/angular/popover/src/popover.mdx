# Popover

<OfficialDocsLink href="https://designsystemet.no/no/components/docs/popover/overview" />

`Popover` vises over andre elementer i grensesnittet og er koblet til et spesifikt element. Den brukes til å vise tilleggsinformasjon, interaktive elementer eller korte forklaringer uten å navigere bort fra siden. `Popover` kan brukes som en utvidet løsning når [`Tooltip`](/docs/komponenter-tooltip--docs) ikke strekker til.

## Bruk

```ts
import { Component } from '@angular/core'
import { Button } from '@ks-digital/designsystem-angular/button'
import { Popover } from '@ks-digital/designsystem-angular/popover'

@Component({
  imports: [Button, Popover]
  template: `
    <button ksd-button popovertarget="my-popover">
      Enkel popover
    </button>
    <div ksd-popover id="my-popover"> Her er det noe innhold </div>
  `,
})
export class App {}
```
