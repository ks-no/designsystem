# Switch

<OfficialDocsLink href="https://designsystemet.no/no/components/docs/switch/overview" />

`Switch` brukes til å gi brukeren et valg mellom to alternativer. Bryteren kan enten slås av eller på og skal alltid være innstilt med et standardvalg

<Primary />
<Controls />

## Bruk

```ts
import { Input, Field, Label } from '@ks-digital/designsystem-angular/forms'

@Component({
  imports: [Field, Label, Input],
  template: `
    <ksd-field>
      <ksd-label>Min switch</ksd-label>
      <input ksd-input type="checkbox" role="switch" />
    </ksd-field>
  `,
})
export class App {}
```

**Gruppering**
