`Radio` er et alternativ brukeren kan velge. Bruk flere `Radio` for Ć„ vise en liste med alternativer. Brukerne kan bytte mellom alternativene, men kan kun velge ett.

## Bruk

Bruk [`Fieldset`](/docs/komponenter-fieldset--docs) for Ć„ visuelt gruppere flere alternativer.

```ts
import {
  Input,
  Field,
  Fieldset,
  FieldsetDescription,
  FieldsetLegend,
  Label,
} from '@ks-digital/designsystem-angular/forms'

@Component({
  imports: [Fieldset, Field, FieldsetDescription, FieldsetLegend, Label, Input],
  template: `
    <fieldset ksd-fieldset>
      <legend ksd-fieldset-legend>Er du over 18 Ć„r?</legend>

      <ksd-field>
        <input ksd-input type="radio" name="alternativ" value="ja" />
        <ksd-label>Ja</ksd-label>
      </ksd-field>

      <ksd-field>
        <input ksd-input type="radio" name="alternativ" value="nei" />
        <ksd-label>Nei</ksd-label>
      </ksd-field>
    </fieldset>
  `,
})
export class App {}
```

Felter med `readonly`-attributtet er med i tabrekkefĆølgen. Brukerne kan kopiere innholdet men ikke redigere det. Informasjon blir med nĆ„r skjemaet sendes inn.
