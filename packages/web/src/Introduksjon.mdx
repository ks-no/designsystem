# Web 👷

Noen komponenter og mønster finnes ikke i Designsystemet, men er nødvendige for KS Digital.

Dette kan f.eks være mer sammensatte komponenter som er for spesifikke for Designsystemet.

Vi jobber med å kartlegge og prioritere hvilke komponenter og mønstre som hører hjemme her og hva som skal videreføres fra gammelt designsystem.

**Komponentene skal kunne brukes i alle løsninger, uavhengig av teknologi.**
